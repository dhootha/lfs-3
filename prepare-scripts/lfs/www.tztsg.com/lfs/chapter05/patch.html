5.26. Patch-2.6.1

Patch 软件包包含一个根据“patch”文件来修改或建立文件的程序。补丁文件通常是用 diff 程序创建的。

预计编译时间: 少于 0.1 SBU
所需磁盘空间: 1.9 MB

5.26.1. 安装 Patch

为编译 Patch 做准备：

./configure --prefix=/tools

编译这个软件包：

make

现在编译完成了 和前面讨论的一样，在本章，因为是临时工具，我们不强制您运行测试组件。 如果一定要运行 Patch 测试系统， 可执行下面的命令： 

make check

安装这个软件包：

make install

这个软件包的详细内容位于 Section 6.51.2, “Contents of Patch.”
