5.24. M4-1.4.13

M4 软件包包含一个宏处理器。

预计编译时间: 0.2 SBU
所需磁盘空间: 11.6 MB

5.24.1. 安装 M4

为编译 M4 做准备：

./configure --prefix=/tools

编译这个软件包：

make

现在编译完成了 和前面讨论的一样，在本章，因为是临时工具，我们不强制您运行测试组件。 如果一定要运行 M4 测试系统， 可执行下面的命令： 

make check

安装这个软件包：

make install

这个软件包的详细内容位于 Section 6.24.2, “Contents of M4.”
