6.48. Make-3.81

Make 软件包包含一个编译软件包的程序。

预计编译时间: 0.3 SBU
所需磁盘空间: 9.7 MB

6.48.1. 安装 Make

首先修复一些自上一版发布以来已经确定的问题：

patch -Np1 -i ../make-3.81-upstream_fixes-1.patch

为编译 Make 做准备：

./configure --prefix=/usr

编译这个软件包：

make

要测试结果，请运行：

make check

安装这个软件包：

make install

6.48.2. Make 的内容

安装的程序: make

简要描述

make

自动地确定一个软件包的哪些片段需要（重新）编译， 然后执行相关的命令。
