6.18. Pkg-config-0.23

pkg-config 包含一个工具，它在配置和编译可执行文件的过程中，通过include目录和/或 library目录构建工具。

预计编译时间: 0.3 SBU
所需磁盘空间: 11.5 MB

6.18.1. 安装 Pkg-config

为编译 Pkg-config 做准备：

./configure --prefix=/usr

编译这个软件包：

make

要测试结果，请运行：

make check

安装这个软件包：

make install

6.18.2. Pkg-config 的内容

安装的程序: pkg-config

简要描述

pkg-config
