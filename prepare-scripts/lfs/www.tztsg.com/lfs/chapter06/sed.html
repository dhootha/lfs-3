6.17. Sed-4.2.1

Sed 软件包包含一个流编辑器。

预计编译时间: 0.2 SBU
所需磁盘空间: 8.3 MB

6.17.1. 安装 Sed

为编译 Sed 做准备：

./configure --prefix=/usr --bindir=/bin --htmldir=/usr/share/doc/sed-4.2.1

新配置选项的含义：

--htmldir

设置HTML文档的安装目录。

编译这个软件包：

make

生成 HTML 文档： 

make html

要测试结果，请运行：

make check

安装这个软件包：

make install

安装 HTML 文档： 

make -C doc install-html
