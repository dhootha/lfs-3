6.14. MPFR-2.4.2

MPFR 软件包包含多精度数学函数。

预计编译时间: 1.1 SBU 包括测试
所需磁盘空间: 27.1 MB 包括测试

6.14.1. 安装 MPFR

为编译 MPFR 做准备：

./configure --prefix=/usr --enable-thread-safe

编译这个软件包：

make

重要

本节的 MPFR 测试套件很关键。在任何情况下都不要省略这一步。

对结果进行测试，确保 148 个测试全部通过：

make check

安装这个软件包：

make install

安装软件文档： 

make html
mkdir -pv /usr/share/doc/mpfr-2.4.2
find . -name \*.html -type f -exec cp -v \{} /usr/share/doc/mpfr-2.4.2 \;
