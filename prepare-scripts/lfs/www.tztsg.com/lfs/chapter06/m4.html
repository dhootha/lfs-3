6.24. M4-1.4.13

M4 软件包包含一个宏处理器。

预计编译时间: 0.4 SBU 包括测试
所需磁盘空间: 14.2 MB

6.24.1. 安装 M4

为编译 M4 做准备：

./configure --prefix=/usr

编译这个软件包：

make

要测试结果，请运行：

make check

安装这个软件包：

make install

6.24.2. M4 的内容

安装的程序: m4

简要描述

m4

m4能够将宏展开并将输入拷贝到输出。宏可以是内嵌的也可以是用户定义的， 并且可以接受很多参数。除了展开宏，m4 还有其它内置的功能， 比如包含引用文件、 运行 Unix 命令、 进行整数运算、 文本操作、 循环等等。 m4 可以被用作一个编译器的前端或作为自身的一个宏处理程序。
